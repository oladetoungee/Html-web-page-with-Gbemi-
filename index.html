HTML Basics

  1. HTML stands for HyperText Markup Language
  2. HTML is used to create and structure contents of web pages
  3. HTML uses tags to define its elements

HTML Tags

Here are some commonly used HTML tags:

  • <h1> - <h6> : Headings
  • <p> : Paragraph
  • <img> : Image
  • <a> : Anchor
  • <ul> : Unordered List
  • <ol> : Ordered List

HTML Attributes

HTML tags can have attributes that provides additional information about our elements

  1. class
  2. id
  3. src
  4. href
  5. style
